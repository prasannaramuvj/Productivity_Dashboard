# Productivity Dashboard

A React-based productivity tracker that helps users manage tasks,
job applications, coding practice, and projects.

## Features
- Dashboard with counts
- To-do list CRUD
- Job application tracker
- Coding tracker with charts
- Project tracker

## Tech Stack
React
JavaScript
HTML
CSS
